##Attune
Personalized Emergency Broadcast System — New York City

When a disaster hits New York City, emergency alerts go out to everyone at once. The same message. The same language. The same assumption that whoever is listening can read, understand, and act. But for an 82-year-old woman with dementia living alone in Chinatown, a flood warning in English means nothing. For a man in the Bronx who is bedridden and speaks only Spanish, an evacuation order is not an instruction — it is just noise. For a grandmother in Flushing with severe hearing loss and PTSD, a sudden alarm doesn't trigger action. It triggers panic.
These are not edge cases. They are the people most likely to be left behind.

#Attune is an emergency broadcast system built for them.
It takes official alerts from the NYC Emergency Management system and transforms them — into calm and human messages delivered in the right language, at the right pace, for the right person. Each message is generated by AI, guided by clinical reasoning that considers cognition, mobility, language, sensory ability, and psychological state. It is then converted to audio and sent directly to a bedside radio device — a small Raspberry Pi speaker that lives in the person's room and speaks to them by name.

#How it works
An operator receives an official NYC alert and processes it through the Attune dashboard. The system identifies which residents are in the affected area, generates a personalized audio message for each person, and broadcasts it to their bedside device — all within minutes. Every message ends with the name of someone the person knows and trusts. Because in a crisis, what most people need to hear is not instructions. It is: someone is coming.

#Built with
OpenAI API — GPT-4o-mini for script generation, TTS for audio synthesis
Netlify Functions — serverless API proxy
Raspberry Pi — bedside audio device
NYC Open Data API — real-time emergency alert feed

#The people it's for
Attune currently holds profiles for 20 elderly residents across all five NYC boroughs, spanning 12 languages including Cantonese, Russian, Bengali, Haitian Creole, Korean, and Vietnamese. Each profile captures not just demographics, but the specific cognitive, physical, sensory, and emotional context that shapes how that person needs to be spoken to.
Because a good emergency system doesn't just reach people. It reaches them in a way they can actually receive.
